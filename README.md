# 🏛️ Accessible Government Procedures and Legal Guidance Platform

An AI-powered government portal that helps citizens understand complex legal and official documents in simple, easy-to-read language.

## 🚀 Features

- 📄 Upload and process PDFs
- 🖼️ Extract text from images using OCR (Tesseract.js)
- 🤖 AI-powered explanation using Groq API (LLaMA models)
- 🧠 Simplified Summary + Step-by-Step Explanation
- 🌐 Multi-language translation support
- 🔊 Text-to-Speech (voice output with speed control)
- 📜 History tracking of processed documents
- 📋 One-click copy output

## 🛠️ Tech Stack

- HTML, CSS, JavaScript
- PDF.js for PDF parsing
- Tesseract.js for OCR
- Groq API (LLaMA 3)
- LibreTranslate API
- Web Speech API

## 📌 How It Works

1. Upload a PDF, image, or type text
2. System extracts content automatically
3. AI processes and simplifies the document
4. Output is shown in structured HTML format
5. User can translate, listen, or copy results

## 🎯 Purpose

To improve accessibility and transparency by helping citizens easily understand legal documents, policies, and government notices without confusion.
